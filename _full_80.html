How to play

  • W A S D or Arrows — Move
  • Space or Click — Cast a fireball (costs 10 MP)
  • 1–3 — Use a potion from the inventory
  • Esc — Pause / resume

Tips

  • Clear every enemy in a room to unlock its doors.
  • Boss rooms hold the level boss — defeat them to advance.
  • From level 3 onwards a timer drains: kill the boss before it hits zero or it's game over.
  • Pickups: Health heals 30 HP, Strength doubles damage for 10s, Invincibility blocks damage for 10s.

About

This is a portfolio web port of a libGDX/Java dungeon crawler. The original Java project lives in core/.

Tech

  • Vanilla HTML / CSS / JavaScript
  • HTML5 Canvas, no engine
  • Procedural pixel sprites (no external assets)
  • Local leaderboard via localStorage

Controls

  • Move: WASD / Arrows
  • Shoot: Space / Click
  • Pause: Esc
